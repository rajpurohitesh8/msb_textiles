# MSB Textiles

Premium fabric store based in Kantin Kwari, Kano, Nigeria. Specializing in Sarkinmenlace, Getzner, Swiss Atiku & Cashmere fabrics — wholesale & retail.

## Tech Stack

- React 18 + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- React Router
- TanStack Query

## Getting Started

```bash
npm install
npm run dev
```
